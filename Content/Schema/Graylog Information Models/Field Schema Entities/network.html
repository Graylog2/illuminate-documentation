Network Fields

field field_type description example_values
network_application keyword/lowercase Application name - Facebook, etc. facebook, instagram
network_bytes long Bytes transferred during a connection, may be calculated by summing bytes sent/received (source_bytes_sent/destination_bytes_sent) - some vendors may report this as packet_length. 71238
network_community_id keyword See: https://github.com/corelight/community-id-spec.
network_connection_duration keyword Duration of time a network connection was established. 0:23:45
network_connection_uid keyword Unique identifier value for a network connection. CMdzit1AMNsmfAIiQc
network_data_bytes long Total bytes of the data payload. 71238
network_direction keyword Indicates the direction of the observed network flow. Must be either inbound or outbound, this should be mapped to these values if vendors provide network direction differently. inbound, outbound, lateral
network_forwarded_ip ip 10.1.2.3, fe80:5cc3:11:4::2c
network_header_bytes long Total bytes of packet header information. 71238
network_iana_number integer https://www.iana.org/assignments/protocol-numbers/protocol-numbers.xhtml. 6, 17, 41
network_icmp_type keyword https://www.iana.org/assignments/icmp-parameters/icmp-parameters.xhtml. echo, time exceeded
network_inner keyword TBD.
network_interface_in keyword/lowercase Name of interface traffic receiving traffic. gi0/1
network_interface_out keyword/lowercase Name of interface traffic sending traffic. gi0/1
network_ip_version keyword IPv4 or IPv6. 4, 6
network_name keyword TBD.
network_packets long Count of packets transferred during a connection, may be calculated by summing packets sent/received (source_packets_sent/destination_packets_sent). 71238
network_protocol keyword/lowercase Protocol names, preferably from the Keyword column in https://www.iana.org/assignments/protocol-numbers/protocol-numbers.xhtml. ipv4, ipv6, icmp
network_transport keyword/lowercase transport layer protocol of packet/connection. udp, tcp
network_tunnel_type keyword/lowercase tunnel type. gre, ipsec
network_tunnel_duration long time in seconds for tunnel duration. 2093847
