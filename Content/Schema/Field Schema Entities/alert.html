Alert Fields

Event log messages sent to Graylog indicating unusual and potentially dangerous or noteworthy activity was detected.

field field_type description example_values
alert_definitions_version keyword Version or identification value that indicates the version a collection of signatures (A/V, etc.) is in use. 2020.1, 4092348
alert_category keyword Vendor-provided category assignment. malware, trojan, ransomware
alert_indicator keyword A filename, URL, packet snippet or other artifact that is related to the event that caused the alert to be generated. malware.exe, http://badsite
