[Introduction]

Supported Controls

  • This Pack current supports a subset of NIST 800-53 AC-2 controls (Access Management)

Requirements

  • Graylog Server 6.0 or later
  • Graylog Illuminate 5.1.0 or later
  • Microsoft Windows event logs delivered to Graylog and processed by Illuminate

Hint: This version of the compliance content has only been tested with Windows content. The content in this pack may include logs from sources other than Windows but only the Windows source has been reviewed for completeness.

Using This Technology Pack

GIM Categorization

The The Graylog Compliance content pack utilizes events that have been categorized using Graylog Information Model (GIM) categorization codes and is not restricted to a specific product.

Lookup Tables

The The Graylog Compliance content pack utilizes lookups that are part of Illuminate Core. These lookups are used by Illuminate Core to apply category data to events having one of the user fields user_name, source_user_name, or target_user_name, creating or adding the fields user_category, source_user_category, or , respectively.
