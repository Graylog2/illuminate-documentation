Sigma Updates

Download Sigma Rule Files

With the release of Graylog 5.2, we have added a new capability that allows you to download Sigma rule files individually or in bulk.

To download a single rule, simply click on the ellipsis next to the desired rule and choose the Download option.

If you want to download multiple rules at once, select the rules you want to download and then click on the Bulk Actions button. From the drop-down menu, select the Download option.

Once you initiate the download, your browser will automatically begin the process. It is important to note that if you choose multiple files, they will be downloaded as a .zip file containing the rules YAML files.

Direct Import of Sigma Files

Along with being able to import Sigma rules from Git repositories, rules can now be imported directly from the file system. To open the upload window, select Upload Rules from the Add Rules drop-down.

In this window both individual Sigma YAML files as well as .zip files containing multiple YAML files can be selected. Each individual rule can have its streams, notifications, and search configuration modified by clicking on the corresponding file in the Selected Files list. A configuration can be applied to all rules using the Use these settings on all rules button. All rules uploaded in a .zip file will be given the same configuration.

On confirmation, all individual rules that were successfully imported will be removed from the Selected Files list. Any rule that failed to import will remain in the list and a reason for the failure will be displayed.


The same process applies in the case of a ZIP file which has invalid rules that cannot be imported, all valid rules will be imported, the failures will be displayed, and the ZIP file will remain in the list.
